# 13.6 Handling Monster Special Abilities / 《 Web 》

## Statements

- Spiders, etc. can generate sticky webs. If a character inadvertently steps into it, he or she must make a resistance roll using _adventurer level + agility bonus_ as the baseline score, and if he or she fails they’ll be caught in the web. To escape from this state, you must make a resistance roll using _adventurer level + strength bonus_ as the baseline score, similar to escaping from _capture_ . Otherwise, you’ll suffer a penalty based on the strength of the web. _(Sword World RPG - Complete Edition.pdf (source-range-870a16bc-06323))_
