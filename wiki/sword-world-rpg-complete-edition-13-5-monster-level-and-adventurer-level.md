## Source record

This section explains how monster levels relate to adventurer levels and provides guidance on encounter scaling.

## Key supported claims

- A monster level’s score is set so that if it equals a character’s adventurer level, the character will be stronger. (raw/Sword World RPG - Complete Edition.pdf p.234-234)
- To model monsters like featherfolk, mermen, dark elves, noble lizardmen, and centaurs as characters, the sub‑ability scores are calculated as shown. (raw/Sword World RPG - Complete Edition.pdf p.234-234)
- Encounter scales: Level 5‑6—too spicy for a novice; Level 7‑8—difficult unless moderate; Level 9‑10—city‑wide crisis; Level 11‑13—nation‑wide crisis; Level 14‑15—Alecrast‑wide crisis. (raw/Sword World RPG - Complete Edition.pdf p.234-234)
- If a monster is 1‑2 levels higher, it matches the character; if 3 levels higher, it is formidable. (raw/Sword World RPG - Complete Edition.pdf p.234-234)
- When added to a scenario, enemies same or one level lower appear frequently; 2‑3 levels higher appear at the climax. (raw/Sword World RPG - Complete Edition.pdf p.234-234)
