# When an Adventurer Casts Magic on Adventurers

From [[sword-world-rpg-complete-edition]].

## Statements

- The following procedure is used to make a check when data created for a character casts magic on adventurers, and vice versa. _(Sword World RPG - Complete Edition.pdf (source-range-de1806a5-01184))_
- First, the side casting the magic rolls 2D. The final score (magic power + 2D) must then be determined this one time only. Each resisting side must then make a mental power resistance roll, using that determined final score as the target score. _(Sword World RPG - Complete Edition.pdf (source-range-de1806a5-01185))_
