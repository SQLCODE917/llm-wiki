## Source record

JavaScript functions are values that evaluate expressions, return values, and have identity semantics.

## Key supported claims

- - Functions are values that can be invoked, not just primitive types. (raw/javascriptallonge.pdf p.30-43)
- - Applying a function to arguments might produce a value, analogous to arithmetic operations. (raw/javascriptallonge.pdf p.30-43)
- - The simplest function syntax is `() => 0`, returning zero. (raw/javascriptallonge.pdf p.30-43)
- - In REPLs, functions print as `[Function]`; the representation can vary by environment. (raw/javascriptallonge.pdf p.30-43)
- - Functions, whether arrow or constructor, are full objects internally. (raw/javascriptallonge.pdf p.30-43)
