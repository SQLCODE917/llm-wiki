## Source record

JavaScript Allongé is a book that explores the fundamentals of JavaScript, focusing on functions, scope, and closures.

## Key supported claims

- Functions are values that can be part of expressions, returned from other functions, and so forth (raw/javascriptallonge.pdf p.78-78).
- Functions are reference values and are applied to arguments (raw/javascriptallonge.pdf p.78-78).
- Fat arrow functions have expressions or blocks as their bodies, whereas function keyword functions always have blocks as their bodies (raw/javascriptallonge.pdf p.78-78).
- JavaScript uses const to bind values to names within block scope, and function declarations to bind functions to names within function scope (raw/javascriptallonge.pdf p.78-78).
- Function application creates a scope, and blocks also create scopes if const statements are within them (raw/javascriptallonge.pdf p.78-78).
