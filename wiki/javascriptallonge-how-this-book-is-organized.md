## Source record

JavaScript Allongé is organized to introduce new aspects of programming with functions in each chapter, explaining exactly how JavaScript works. Code examples within each chapter are small and emphasize exposition rather than serving as patterns for everyday use. Following some of the chapters are a series of recipes designed to show the application of the chapter's ideas in practical form.
