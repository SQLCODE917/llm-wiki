## Source record

JavaScript Allongé is organized to introduce new aspects of programming with functions in each chapter, with code examples that emphasize exposition rather than everyday use patterns. Some chapters are followed by practical recipes that apply the chapter's ideas.

## Key supported claims

- JavaScript Allongé introduces new aspects of programming with functions in each chapter, explaining exactly how JavaScript works (raw/javascriptallonge.pdf p.11-12).
- Code examples within each chapter are small and emphasize exposition rather than serving as patterns for everyday use (raw/javascriptallonge.pdf p.11-12).
- Following some of the chapters are a series of recipes designed to show the application of the chapter's ideas in practical form (raw/javascriptallonge.pdf p.11-12).
