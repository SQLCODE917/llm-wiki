# Should You Use a Target Score Check or a Difficulty Check?

From [[sword-world-rpg-complete-edition]].

## Statements

- There are two types of success rolls: those that use a target score for the check and those that use a difficulty for the check. The game master must decide which of these should be used to make the success roll. _(Sword World RPG - Complete Edition.pdf (source-range-de1806a5-03933))_
- The standard for deciding is whether the character can tell if the action succeeded or failed . For example, if you want to jump over a cliff, success or failure is clear. If you succeed, you can jump over it, and if you fail, you will fall. Checks for actions with clear successes or failures are made using target scores . _(Sword World RPG - Complete Edition.pdf (source-range-de1806a5-03934))_
- An example of something where success or failure cannot be determined is finding a trap. When a trap is not found, a character has no way of knowing whether there really are no traps or if it's because they couldn't find it. The same goes for when you're sneaking up on a sleeping monster. Even though it looks like it's a success, the monster may actually be pretending to be asleep. _(Sword World RPG - Complete Edition.pdf (source-range-de1806a5-03935))_
- These actions result in a difficulty check . If you can fail on a success roll but still have it appear to be a success, then the success roll should be made as a difficulty check. _(Sword World RPG - Complete Edition.pdf (source-range-de1806a5-03936))_
