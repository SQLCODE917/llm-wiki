Chapter on the Maybe pattern and its implementation in JavaScript

## Key supported claims

- Function decorator for handling null/undefined values, borrowed from Haskell's maybe monad, Ruby's andand, and CoffeeScript's existential method invocation (raw/javascriptallonge.pdf p.86-87).
- Maybe plays nicely with instance methods (raw/javascriptallonge.pdf p.86-87).
- The maybe decorator reduces logic of checking for nothing to a function call (raw/javascriptallonge.pdf p.86-87).
