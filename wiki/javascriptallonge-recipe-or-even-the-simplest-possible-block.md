# Or even: > the simplest possible block

## The Simplest Block

A block can be the simplest possible construct with no statements inside. _(raw/javascriptallonge.pdf (p. 34))_ 

## Block Evaluation

When a block has no statements, evaluating it returns undefined. _(raw/javascriptallonge.pdf (p. 34))_ 

## Block Syntax

A block is defined as zero or more statements separated by semicolons. _(raw/javascriptallonge.pdf (p. 34))_
