# Or even: > the simplest possible block

## The Simplest Possible Block

The simplest possible block in JavaScript is an empty block, which contains zero statements separated by semicolons. _(raw/javascriptallonge.pdf (p. 34))_ 

## Execution of Empty Blocks

When an empty block is evaluated, it returns undefined, as demonstrated by the code example ((() => {})()). _(raw/javascriptallonge.pdf (p. 34))_
