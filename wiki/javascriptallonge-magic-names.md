Chapter on magic names in JavaScript Allongé

## Key supported claims

- JavaScript binds argument values to function argument names during execution (raw/javascriptallonge.pdf p.74-74).
- JavaScript also binds values to some "magic" names in addition to user-defined arguments (raw/javascriptallonge.pdf p.74-74).
- Magic names are special identifiers bound during function execution (raw/javascriptallonge.pdf p.74-74).
