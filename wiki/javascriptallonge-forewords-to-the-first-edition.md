## Source record

Forewords to the first edition of JavaScript Allongé by Michael Fogus and Matthew Knox, providing context and endorsement for the work.

## Key supported claims

- Michael Fogus was skeptical about the pay-what-you-want ebook model but was humbled by the quality of JavaScript Allongé, finding it a great way to write and distribute books (raw/javascriptallonge.pdf p.14-15).
- The Leanpub model enabled an efficient read-feedback-change process, resulting in a quality book that continues to improve with reader feedback (raw/javascriptallonge.pdf p.14-15).
- Matthew Knox notes that JavaScript Allongé provides a gentle start before showing the full depth of JavaScript, comparing it to a Cafe Allongé that is over too soon (raw/javascriptallonge.pdf p.14-15).
