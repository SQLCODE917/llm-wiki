## Source record

JavaScript has a literal syntax for creating an array using the [ and ] characters. Array literals are expressions, and arrays are reference types.

## Key supported claims

- Array literals are expressions, and arrays are reference types. (raw/javascriptallonge.pdf p.101-102)
- Each time an array literal is evaluated, a new, distinct array is created, even with identical elements. (raw/javascriptallonge.pdf p.101-102)
- JavaScript uses [ and ] characters for array literal syntax. (raw/javascriptallonge.pdf p.101-102)
- Array literals can be nested, creating arrays with multiple levels of nesting. (raw/javascriptallonge.pdf p.101-102)
