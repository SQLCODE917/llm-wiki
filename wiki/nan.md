NaN (Not-a-Number) is a special value in JavaScript indicating invalid numerical operations. Key characteristics include:

- Generated by invalid arithmetic operations [raw/javascriptallonge.pdf p.30-43]
- Not equal to itself (`NaN !== NaN`)
- Distinct from `undefined` and `null`

See [[javascriptallonge-as-little-as-possible-about-functions-but-no-less]] for behavior examples and [[void]] for related operators.
