# 8. Game Master Screen

From [[sword-world-rpg-complete-edition]].

## Statements

- The Sword World RPG Game Master Screen is a collection of charts (or tables) that you will need in order to play this game. This can be useful for smooth game progress. _(Sword World RPG - Complete Edition.pdf (source-range-de1806a5-00126))_
