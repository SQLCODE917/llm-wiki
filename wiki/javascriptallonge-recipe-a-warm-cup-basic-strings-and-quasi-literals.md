# A Warm Cup: Basic Strings and Quasi-Literals

## String Concatenation

JavaScript supports string concatenation using the + operator, as demonstrated by the expression 'fu' + 'bar' which evaluates to 'fubar'. _(raw/javascriptallonge.pdf (p. 203))_ 

## Understanding Strings

In JavaScript, strings are fundamental data types used to represent text, and they are literals like 'fubar' or 'fizzbuzz'. _(raw/javascriptallonge.pdf (p. 203))_ 

Special characters in string literals can be included using escape sequences, such as \n for inserting a newline character. _(raw/javascriptallonge.pdf (p. 203))_ 

An expression is any valid unit of code that resolves to a value, according to the Mozilla Development Network. _(raw/javascriptallonge.pdf (pp. 202-203))_ 

## String Operators and Manipulation

There are operators available for working with strings, with the + operator being the most common for concatenating strings. _(raw/javascriptallonge.pdf (p. 203))_ 

String manipulation is extremely common in programming, as writing is a big part of what makes us human, and strings are how JavaScript and most other languages represent writing. _(raw/javascriptallonge.pdf (p. 203))_
