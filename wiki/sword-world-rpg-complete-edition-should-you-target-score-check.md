# Should You Target Score Check

What [[sword-world-rpg-complete-edition]] covers about should you target score check:

## Statements

### Chapter 9: / Setting Target Scores and Difficulties / Should You Use a Target Score Check or a Difficulty Check?

- There are two types of success rolls: those that use a target score for the check and those that use a difficulty for the check. The game master must decide which of these should be used to make the success roll. _(Sword World RPG - Complete Edition.pdf (source-range-0b9dec90-05607))_

- The standard for deciding is whether the character can tell if the action succeeded or failed . For example, if you want to jump over a cliff, success or failure is clear. If you succeed, you can jump over it, and if you fail, you will fall. Checks for actions with clear successes or failures are made using target scores . _(Sword World RPG - Complete Edition.pdf (source-range-0b9dec90-05608))_

- These actions result in a difficulty check . If you can fail on a success roll but still have it appear to be a success, then the success roll should be made as a difficulty check. _(Sword World RPG - Complete Edition.pdf (source-range-0b9dec90-05610))_
