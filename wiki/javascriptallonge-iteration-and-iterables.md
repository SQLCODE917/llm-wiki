## Source record

Chapter explains JavaScript collection iteration, functional iterators, iterables, and operations on ordered collections.

## Key supported claims

- Collections in JavaScript can model sets of items, and iterating over them is achieved via iterator methods. (raw/javascriptallonge.pdf p.206-223)
- A functional iterator can be defined on an object, returning a function that yields successive values, and this approach might simplify certain algorithms. (raw/javascriptallonge.pdf p.206-223)
- The Symbol.iterator protocol enables for...of loops and spread syntax, but spreading infinite iterables may cause infinite loops. (raw/javascriptallonge.pdf p.206-223)
