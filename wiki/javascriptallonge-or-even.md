# Or even:

This section discusses the use of parentheses in JavaScript to group expressions, particularly in the context of arrow functions.

```
() => ( 1 + 1, 2 + 2 )
```

This example shows an arrow function that evaluates two expressions, returning the result of the second one. The parentheses are used to group the expressions, which is a common pattern in JavaScript for controlling the order of operations and ensuring that the last expression's result is returned.

## Key Concepts

- **Parentheses in Arrow Functions**: The use of parentheses to group expressions within arrow functions.
- **Expression Evaluation**: Understanding how JavaScript evaluates multiple expressions within a function body.
