Node.js is a JavaScript runtime built on Chrome's V8 engine, enabling JavaScript to run outside web browsers. Key features include:

- REPL environment that displays [Function] for functions [raw/javascriptallonge.pdf p.30-43]
- Asynchronous I/O model
- NPM package ecosystem

See [[javascript]] for language characteristics and [[javascriptallonge-as-little-as-possible-about-functions-but-no-less]] for function behavior in this environment.
