## Source record

The simplest possible block in JavaScript Allongé is a block with no statements, which returns the result of evaluating that block.

## Key supported claims

- A block with no statements returns undefined (raw/javascriptallonge.pdf p.34-34).
- A block can be placed to the right of an arrow (raw/javascriptallonge.pdf p.34-34).
- A block is a sequence of statements separated by semicolons (raw/javascriptallonge.pdf p.34-34).
