## Source record

JavaScript Allongé, the 'Six' Edition is both a comprehensive tour of its features and a rich collection of techniques for making better use of them.

## Key supported claims

- JavaScript Allongé, the 'Six' Edition is both a comprehensive tour of its features and a rich collection of techniques for making better use of them (raw/javascriptallonge.pdf p.13-13).
- With ECMAScript 6, JavaScript has become much larger as a language (raw/javascriptallonge.pdf p.13-13).
- Better syntax for features that already exist (e.g. via libraries), such as classes and modules (raw/javascriptallonge.pdf p.13-13).
- New functionality in the standard library, such as new methods for strings and arrays, Promises, Maps and sets (raw/javascriptallonge.pdf p.13-13).
- Completely new features, such as Generators, proxies and WeakMaps (raw/javascriptallonge.pdf p.13-13).
