# true

The boolean value `true` is a fundamental concept in JavaScript. It represents one of the two logical values in the language, the other being `false`. In JavaScript, `true` is used in conditional statements, logical operations, and to control the flow of programs.

In the context of this book, `true` is discussed as a core part of JavaScript's boolean type, which is used in various expressions and control structures. The value `true` can be used in logical operations such as `&&` (AND), `||` (OR), and `!` (NOT).

For example, in JavaScript:
```javascript
//=> true
```
This demonstrates that `true` is a valid and commonly used value in JavaScript code.
