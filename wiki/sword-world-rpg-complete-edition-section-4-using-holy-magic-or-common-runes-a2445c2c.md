# 4. Using Holy Magic or Common Runes

From [[sword-world-rpg-complete-edition]].

## Statements

- Even while grappling, you can use holy magic so long as you can speak. It's not possible while being strangled , but if you're merely being pinned down , you can use holy magic. Common runes require you to not only be able to speak, but to also be able to use a catalyst. It's usually a ring, so you can use it. _(Sword World RPG - Complete Edition.pdf (source-range-de1806a5-08971))_
- Ancient magic, spirit magic, and spellsongs cannot be used. _(Sword World RPG - Complete Edition.pdf (source-range-de1806a5-08972))_
- Common runes require you to not only be able to speak, but to also be able to use a catalyst. _(Sword World RPG - Complete Edition.pdf (source-range-de1806a5-08971))_
