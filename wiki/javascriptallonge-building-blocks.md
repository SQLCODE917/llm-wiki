# Building Blocks

## JavaScript's Flexibility and Challenges

JavaScript code can become difficult to follow when functions are nested within other functions. _(raw/javascriptallonge.pdf (p. 71))_ 

## Strengths and Weaknesses of JavaScript

JavaScript allows for a great deal of flexibility in how code is written. _(raw/javascriptallonge.pdf (p. 71))_ 

This flexibility can lead to code that is hard to manage and maintain. _(raw/javascriptallonge.pdf (p. 71))_ 

## Code Structure and Patterns

JavaScript code often includes a mix of control structures and return statements. _(raw/javascriptallonge.pdf (p. 71))_ 

Understanding basic patterns can help organize code more effectively. _(raw/javascriptallonge.pdf (p. 71))_
