## Source record

Summary of the Building Blocks chapter from JavaScript Allongé, covering the concepts of spaghetti code, the strength and weakness of JavaScript, and the importance of basic building blocks for structuring code.

## Key supported claims

- The strength of JavaScript is that you can do anything (raw/javascriptallonge.pdf p.71-71).
- There are ifs, fors, returns, everything thrown higgledy piggledy together (raw/javascriptallonge.pdf p.71-71).
- Although you needn't restrict yourself to a small number of simple patterns, it can be helpful to understand the patterns so that you can structure your code around some basic building blocks (raw/javascriptallonge.pdf p.71-71).
