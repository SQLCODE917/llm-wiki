# Setting Target Score

What [[sword-world-rpg-complete-edition]] covers about setting target score:

## Statements

### Chapter 9: / Setting Target Scores and Difficulties

- To make a success roll, you must decide on the skill to be used , the relevant ability score , and the target score or difficulty . Of these, skill and ability score should be clear immediately as long as you understand the abilities included under each skill. The problem is the target score or difficulty. There is no problem with things that are clearly stipulated in the rules, such as climbing, hit checks in combat, etc. but things that are not will have to be set by the game master each time they are needed. _(Sword World RPG - Complete Edition.pdf (source-range-0b9dec90-05542))_

### Chapter 9: / Setting Target Scores and Difficulties / Should You Use a Target Score Check or a Difficulty Check?

- There are two types of success rolls: those that use a target score for the check and those that use a difficulty for the check. The game master must decide which of these should be used to make the success roll. _(Sword World RPG - Complete Edition.pdf (source-range-0b9dec90-05607))_

- The standard for deciding is whether the character can tell if the action succeeded or failed . For example, if you want to jump over a cliff, success or failure is clear. If you succeed, you can jump over it, and if you fail, you will fall. Checks for actions with clear successes or failures are made using target scores . _(Sword World RPG - Complete Edition.pdf (source-range-0b9dec90-05608))_

**Context:** _(Sword World RPG - Complete Edition.pdf (source-range-0b9dec90-05609))_

> An example of something where success or failure cannot be determined is finding a trap. When a trap is not found, a character has no way of knowing whether there really are no traps or if it's because they couldn't find it. The same goes for when you're sneaking up on a sleeping monster. Even though it looks like it's a success, the monster may actually be pretending to be asleep.
