This section introduces the concept that all values are expressions in JavaScript, using a coffee shop analogy to illustrate the idea.

## Key supported claims

- All values are expressions, as demonstrated by the café Cubano example (raw/javascriptallonge.pdf p.19-20).
- The number 42 is both an expression and a value in JavaScript (raw/javascriptallonge.pdf p.19-20).
- String concatenation with the + operator creates expressions that are not values (raw/javascriptallonge.pdf p.19-20).
