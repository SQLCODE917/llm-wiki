This page summarizes the front matter of *JavaScript Allongé, the “Six” Edition* by Reg “raganwald” Braithwaite. It includes the title, author information, copyright details, and prefaces that set the stage for the book's content.

## Title and Author

- **Title**: JavaScript Allongé, the “Six” Edition
- **Author**: Reg “raganwald” Braithwaite
- **Publisher**: Leanpub
- **Publication Date**: 2017-11-03
- **Book Link**: [JavaScript Allongé on Leanpub](http://leanpub.com/javascriptallongesix)

## Prefaces

The book contains several prefaces that introduce the concepts and purpose of the work:

1. **A Pull of the Lever: Prefaces** - Introduces the metaphor of "Café Allongé" to describe the book's approach to programming with functions. It explains how JavaScript hits a sweet spot in having widely used language features that are well-suited for functional programming.
2. **About JavaScript Allongé** - Clarifies that the book is fundamentally about *programming with functions*. It discusses how JavaScript's features, especially first-class functions with lexical scope, make it ideal for this approach. The "Six" edition updates the content for ECMAScript 2015 (ES6) features.
3. **Why the “Six” Edition?** - Explains that the update to the "Six" edition is due to ECMAScript 2015 introducing new features that make programming with functions easier and more expressive. It also discusses how the new features align with the book's core ideas of composing small, flexible entities.
4. **What JavaScript Allongé is. And isn’t.** - Emphasizes that the book is about thinking about programs and the fundamentals of programming with functions, not about prescriptive best practices. It acknowledges that software development choices are influenced by social, requirement, and tooling considerations.
5. **Foreword to the “Six” edition** - Written by Axel Rauschmayer, it describes the history of ECMAScript 6 (ES6) and its major groups of features, including better syntax for existing features, new functionality in the standard library, and completely new features. It highlights how the book serves as a comprehensive tour of ES6 features and techniques for functional and object-oriented programming.

## Contents Overview

The book is organized into chapters covering various aspects of programming with functions, including:
- Values and expressions
- Basic numbers
- Basic functions
- Recipes with basic functions
- Choice and truthiness
- Composing and decomposing data
- Basic strings and quasi-literals
- Served by the pot: collections
- The golden crema: appendices and afterwords

## Key Concepts

- **Programming with Functions**: The central theme is using functions as the primary building blocks of programs.
- **Functional Programming**: Techniques for writing cleaner, simpler, and less complicated code using functional approaches.
- **ECMAScript 2015 (ES6)**: The update to the "Six" edition focuses on how ES6 features make programming with functions more powerful and expressive.
- **Composition**: The idea of building larger programs from smaller, independent, and composable entities.
- **Flexibility**: The book emphasizes the flexibility of functions and how they can be used to create adaptable and decoupled code.
