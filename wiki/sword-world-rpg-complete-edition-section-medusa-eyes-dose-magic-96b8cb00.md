# **[Medusa Eyes (Dose/Magic)]**

From [[sword-world-rpg-complete-edition]].

## Statements

- This is a clear liquid poison that has a slight odor, so it's used by mixing with alcohol, etc. _(Sword World RPG - Complete Edition.pdf (source-range-870a16bc-04371))_
- Those who fail to resist will have their bodies petrified and become stone statues. _(Sword World RPG - Complete Edition.pdf (source-range-870a16bc-04371))_

### Technical atom 1

**Context:** _(Sword World RPG - Complete Edition.pdf (source-range-870a16bc-04367))_

> This is a small granular poison, and if someone drinks it and fails to resist, they will become a woman if they're a man, and a man if they're a woman. There are no other penalties. Additionally, there are no changes to the magic that shamans can use. If you drink it again, you can return to normal.
