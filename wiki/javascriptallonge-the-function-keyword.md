## Source record

JavaScript does have a syntax for naming a function, we use the function keyword. Until ECMAScript 2015 was created, function was the usual syntax for writing functions.

## Key supported claims

- Function keyword syntax for naming functions (raw/javascriptallonge.pdf p.74-75)
- First magic name is 'this', bound to function's context (raw/javascriptallonge.pdf p.74-75)
- Second magic name is 'arguments', contains all passed arguments (raw/javascriptallonge.pdf p.74-75)
- Arguments is not an array, but an object with integer-like properties (raw/javascriptallonge.pdf p.74-75)
- Most common use of arguments is for variable argument functions (raw/javascriptallonge.pdf p.74-75)
