## Source record

Foreword to the "Six" edition of JavaScript Allongé, discussing ES6 standardization and features.

## Key supported claims

- ECMAScript 6 (ES6) was ratified as a standard on June 17, with origins dating back to the year 2000, after ECMAScript 3 was finished (raw/javascriptallonge.pdf p.13-13).
- ECMAScript 4 was planned to have numerous new features, which would have turned JavaScript into a completely new language, but was abandoned in favor of ES5 and ES6 (raw/javascriptallonge.pdf p.13-13).
- JavaScript Allongé, the 'Six' Edition, is a comprehensive tour of ES6 features and techniques for better use of them, covering functional and object-oriented programming via ES6 code (raw/javascriptallonge.pdf p.13-13).
