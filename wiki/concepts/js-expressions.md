# JavaScript Expressions

## Summary

In JavaScript, an expression is a unit of code that evaluates to a value. Every value in JavaScript is itself an expression. Expressions can be simple like a number or string literal, or complex like function calls or arithmetic operations.

## Source-backed details

| Claim | Evidence |
| --- | --- |
| Every value in JavaScript is considered an expression that evaluates to itself. | [js-allonge:claim_js-allonge_c001_af9ebb2b] |

| Array literals in JavaScript generate distinct objects even when containing identical elements. | [js-allonge:claim_js-allonge_c001_74b510ad] |
