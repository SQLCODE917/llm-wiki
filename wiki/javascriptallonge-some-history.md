## Source record

Lisp, an acronym for LISt Processing, was one of the first high-level programming languages. The CPU's instruction set featured two important macros: CAR fetches 15 bits from the Address part of the Register, while CDR fetches the Contents of the Decrement part of the Register. Lisp's basic data type is the cons cell, two 15-bit values stored in one word. Lists were represented as linked lists of cons cells.
