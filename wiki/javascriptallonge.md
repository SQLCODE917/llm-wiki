# JavaScript, Allongé

*JavaScript, Allongé* is a book that explores functional programming concepts in JavaScript, focusing on functions, closures, scope, and advanced patterns. It covers fundamental topics such as function application, closures, and scope, as well as more advanced concepts like recursion, garbage collection, mutation, and data structures. The book also delves into iterables, generators, and lazy vs eager collections, with practical examples and exercises.
